## Instalação



|                                                                     |                                                                                             |
| :-------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------: |
| <p style="color:DodgerBlue"> Baixar o projeto</p>                   |                                 `git clone link_do_projeto`                                 |
| <p style="color:DodgerBlue"> Baixar as dependências do projeto</p> |                                            `yarn`                                            |
